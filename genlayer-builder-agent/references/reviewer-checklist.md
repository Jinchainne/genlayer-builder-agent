# Reviewer Checklist

Approve only if a reviewer can answer yes to most of these in a few minutes:

- Can I identify the exact GenLayer use case quickly?
- Can I find the file that executes the non-deterministic path?
- Can I see a real read or write path?
- Can I tell what happens during resolution and after execution?
- Does the README explain how to verify the repo?
- Are there tests, scripts, or CI signals that back up the claim?

If too many answers are no, the project is not ready.
